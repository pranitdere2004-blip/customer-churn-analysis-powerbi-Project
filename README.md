# Customer Churn Analysis Using Power BI

## Project Overview
This project analyzes customer churn in a telecom company using Power BI. The goal of the analysis is to identify the key factors that influence customers to leave the company and help businesses improve their customer retention strategies.

The project uses a telecom customer dataset to analyze churn patterns based on service usage, contract type, tenure, and payment methods. An interactive Power BI dashboard was created to visualize churn behavior and provide meaningful business insights.

## Objectives
- Analyze customer churn behavior using historical telecom data.
- Identify key factors influencing churn such as Internet Service, Contract Type, Tenure, and Payment Method.
- Detect high-risk customer segments likely to churn.
- Support business decision-making through interactive dashboards.

## Data Processing
- Imported telecom customer dataset into Power BI.
- Cleaned and prepared the dataset.
- Handled missing values and corrected data types.
- Created calculated columns and measures for churn analysis.
- Prepared the data model for dashboard visualization.

## Dashboard Pages
The Power BI dashboard includes four main pages:

1. **Overview**
   - Total churned customers
   - Revenue loss
   - Tenure-based churn analysis
   - Gender-wise churn distribution

2. **Demographics Analysis**
   - Service usage patterns
   - Internet service distribution
   - Phone and streaming service analysis

3. **Customer Account Analysis**
   - Contract types
   - Payment methods
   - Billing preferences

4. **Churn Risk Analysis**
   - Overall churn rate
   - Average monthly charges
   - Average customer tenure
   - Churn rate by internet service and payment methods

## Key Insights
- Customers using **Fiber Optic internet service** show the highest churn rate.
- **Month-to-month contracts** have significantly higher churn compared to long-term contracts.
- Customers with **short tenure (first year)** are more likely to leave.
- **Manual payment methods** are associated with higher churn risk.

## Business Recommendations
- Improve service quality for fiber optic customers.
- Encourage long-term contracts through loyalty benefits and discounts.
- Focus on customer retention during the first year.
- Promote automatic payment methods to reduce churn risk.
